# Privacy Policy — QueueSaathi

**Last updated: June 26, 2026**

## What We Collect

QueueSaathi collects the following information from patients:

- **Name** — provided by the patient during token booking
- **Phone number** — used to identify the patient and send booking confirmations via WhatsApp

## How We Use It

- To book and manage clinic queue tokens
- To send WhatsApp notifications about token status and turn updates
- To display patient information on the clinic's doctor dashboard

## Data Storage

- Patient data is stored securely in a cloud database
- Data is associated with the clinic where the booking was made
- Daily token records are retained for clinic record-keeping purposes

## Data Sharing

- Patient data is **not sold or shared** with any third party
- Data is only accessible to the clinic staff where the patient booked a token

## Patient Rights

- Patients can cancel their token at any time by replying **CANCEL** on WhatsApp
- For data deletion requests, contact us at the email below
